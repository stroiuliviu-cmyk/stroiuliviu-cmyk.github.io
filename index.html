Ce este Git?

Git este un sistem de control al versiunilor care te ajută să gestionezi fișierele proiectului pe care lucrezi, permițându-ți să urmărești modificările, să colaborezi cu alți dezvoltatori și să gestionezi istoricul proiectului.

Acestea sunt câteva dintre comenzile esențiale pe care le vei folosi pentru a lucra cu Git:

  • git init: Inițializează un depozit Git
  • git add: Adaugă fișiere la staging area
  • git commit: Salvează modificările în istoric
  • git push: Împinge modificările către un depozit remote
  • git pull: Împinge modificările din depozitul remote în local
  • git branch: Lucrează cu ramuri
